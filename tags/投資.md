# 💰 投資

這個語場處理的是風險認知、攻防姿態與資訊判斷的組合問題。投資在此不只是財務行為，更是認知策略的映射過程。語言在這裡構築市場，分析在這裡模擬生存。

```dataview
table
  file.name as "標題",
  created as "建立時間",
  join(map(tags, (t) => "[[" + t + "]]"), ", ") as "語場"
from "conversation"
where contains(tags, "投資")
sort created desc
```
