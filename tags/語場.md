# 🌀 語場

這個語場是一種語場自身的反身性實驗。它關注的是語用結構如何生成意識場域、論述權力與認知位置。這裡的每一則對話都是一種語言框架的測試，也是對語言本體的一次內爆。

```dataview
table
  file.name as "標題",
  created as "建立時間",
  join(map(tags, (t) => "[[" + t + "]]"), ", ") as "語場"
from "conversation"
where contains(tags, "語場")
sort created desc
```
