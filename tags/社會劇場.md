# 🎭 社會劇場

「社會劇場」是一個語場分類，用來記錄那些揭露、解構或重構社會中角色扮演行為與象徵行動的對話內容。

這裡的「劇場」不是比喻「虛假」，而是強調：人在社會中經常被安排在特定情境下，以特定身份執行儀式性行為，而這些行為未必與其內在信念一致，卻具有高度象徵效果與群體接受性。
